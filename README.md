# PPDT
A static web tool for SSB PPDT practice featuring strict 30s perception and 4m writing timers with visual warnings. Runs entirely in the browser with local image processing for privacy. No backend required. Perfect for aspirants to practice time management and simulate real exam conditions.

SSB PPDT Practice Tool
A web-based practice tool for the Picture Perception and Discussion Test (PPDT), designed specifically for aspirants of the Indian Armed Forces (Army, Navy, Air Force) via the SSB Interview process.

This project is a static web application hosted on GitHub Pages. It allows users to practice PPDT with strict timing rules, mirroring the actual SSB environment, without needing any backend server or database.

📖 About The Project
The PPDT is a crucial screening test conducted on Day 1 of the SSB interview. It tests a candidate's perception, analytical thinking, and narrative writing skills under time pressure.

This tool was built to solve the problem of practicing PPDT alone or without a mentor. It enforces strict time limits and provides a distraction-free interface to simulate real exam conditions.

Key Features
Strict Timing Rules:
Perception Phase: 30 seconds to view the hazy picture.
Writing Phase: 4 minutes to write the story.
No Pause/Skip: Once started, the test cannot be paused or stopped, forcing the user to adapt to the pressure.
Visual Warnings:
Timer turns Orange (Pulse) at 30 seconds remaining.
Timer turns Red (Shake) at 10 seconds remaining to indicate closing time.
Local Privacy: All images are processed locally in the browser. No images are uploaded to any server, ensuring complete privacy for the user.
Bulk Upload: Users can upload multiple images at once to practice a full set.
Zero Dependencies: Runs on pure HTML, CSS, and JavaScript. No frameworks or installations required.

🏁 Getting Started
To use this tool, you do not need to install any software. Simply visit the Live Demo link provided above.

How to Use
Open the Link: Open the website in any modern browser (Chrome, Firefox, Edge).
Read Rules: The landing page displays the strict rules of engagement.
Upload Images: Click the upload zone to select one or more images from your device.
Start Practice: Click the "Start Practice" button.
Observe: The first image appears for 30 seconds only. Memorize the details.
Write: After 30 seconds, the image hides. A 4-minute timer starts centered on the screen. Write your story on a piece of paper or notepad.
Repeat: Once the time is up, the next image loads automatically.
💻 Deployment (For Developers)
If you wish to host this tool yourself:

Clone or download the repository.
Ensure the main file is named index.html.
Push to a GitHub repository.
Go to Settings > Pages in your repository.
Set the source to the main branch.
Your site will be live at https://niju53ts02-ui.github.io/PPDT/
📝 Disclaimer
This tool is for practice purposes only. The actual PPDT in SSB involves group discussions and narration, which are not covered in this tool. This tool focuses specifically on the Perception and Story Writing time management aspects.
